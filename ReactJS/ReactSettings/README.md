# React 설치 및 기본세팅
```
1. Node.js 설치 -> https://nodejs.org/en/

2-1. $ npm install
or
2-2. $ npm install -g yarn

3-1.
    npx create-react-app my-app
    cd my-app
    npm start
or
3-2.
    yarn global add create-react-app
    $ yarn create react-app my-app
    cd my-app
    yarn start

```
