- JS에서 기억해야 할 것
  1. Create(생성)
  2. Initialize(초기화)
  3. Use(사용)
  4. Camel case(카멜 표기법)
     - 변수명 중간에 스페이스가 필요하면 스페이스 대신에 다음단어 첫글자를 대문자로 써준다.

  6. ' = ' 하나는 변수를 "할당"하는 것, ' === ' 세개는 완전히 같은지 "확인/체크"하는 것.
  7. const 변수명을 사용할 때는, 보통 대문자로  변수명을 쓴다.
